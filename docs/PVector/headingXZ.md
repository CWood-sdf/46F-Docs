# double headingXZ()
This function returns the heading in the XZ plane. For this, the Z axis is the forward axis.

## Returns
A double that holds the heading in the XZ plane.

## Example
```cpp
PVector v = PVector(1, 0, 1);
v.headingXZ(); // Returns 45
```
