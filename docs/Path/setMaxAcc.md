# Path& setMaxAcc(double maxAcc)
Sets the maximum acceleration of the path

## Parameter
- `maxAcc` - The maximum acceleration of the path in in/s^2 units

## Returns
The path

## Example
```cpp
Path path = Path();

path.setMaxAcc(100);
```
