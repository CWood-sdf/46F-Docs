# void pushBase(T&amp;&amp; val)

Adds the value to the start of the list

## Parameters

- `val` - the value to add

## Example

```cpp
LinkedList<int> l = {3, 4, 5};
l.pushBase(6);
//l now contains {6, 3, 4, 5}
```
