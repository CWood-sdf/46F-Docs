# void pushBack(T&amp;&amp; val)

Adds the value to the end of the list

## Parameters

- `val` - the value to add

## Example

```cpp
LinkedList<int> l = {3, 4, 5};
l.pushBack(6);
//l now contains {3, 4, 5, 6}
```
